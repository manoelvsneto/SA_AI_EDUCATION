# Educational Context

Summary of the educational context and course activities.

| Element | Description |
|---------|-------------|
| **Pedagogical approach** | Project-Based Learning with dialogical expository classes, checkpoints, incremental project, and feedback. |
| **Core content** | SA fundamentals, cloud, documentation, architectural styles, quality, decisions, and *trade-offs*. |
| **Styles and patterns** | Microservices, serverless, BFF, API Gateway, Event-Driven Architecture, Clean Architecture, and Vertical Slice. |
| **Modeling** | Documentation with arc42, C4 Model, UML, and diagrams as code. |
| **Generative AI** | Permitted use as support for implementation, review, and exploration, maintaining student responsibility for quality. |
| **Final delivery** | Application deployed in the cloud and corresponding architectural documentation. |
